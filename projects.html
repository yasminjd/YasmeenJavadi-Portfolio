Stya – AI-Powered Personal Stylist App Available on App Store

Personal Project • April 2025

https://stya.ai

Download on App Store

✓

Summary: Stya is a mobile app that acts as an AI-powered personal stylist. Users upload images of their clothing, and the app generates personalized outfit recommendations using generative AI. It even visualizes how outfits would look on the user — eliminating the need to try them on physically. Future iterations will include agentic AI that shops for users automatically based on their wardrobe, preferences, and lifestyle context.

✓

My Role: Co-founder and lead engineer. I worked on the iOS architecture (Swift), backend integration (Firebase), and GPT-4 prompt engineering trained on fashion stylist research. I also coordinated FASHN API integration for try-on visualization and iterated rapidly using AI-enhanced IDEs like Windsurf and Cursor.

✓

Tools & Tech: Frontend: Swift, SwiftUI | Backend: Firebase Authentication, Firestore, Storage | AI APIs: OpenAI GPT-4.1, FASHN API | Frameworks: Vision, Swift Package Manager | IDEs: Windsurf, Cursor AI IDE

✓

Outcome: Successfully launched on App Store with 5.0-star rating, actively maintaining with regular updates (latest v1.3), growing user base providing positive feedback on time-saving and style discovery features

✓

Personal Motivation: Stya solves a real frustration I personally face as a fashion-conscious shopper — spending time every day trying to figure out what to wear, despite having a full closet. After hearing similar experiences from friends and seeing how widespread this problem was (especially among women), we built Stya to offer a smarter way to dress with confidence using AI.
