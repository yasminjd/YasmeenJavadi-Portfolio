About the Project

Developed a multi-class classification system to predict obesity levels in individuals based on lifestyle habits and physical attributes. The model achieves 95% accuracy and can classify individuals into 7 distinct weight categories, from underweight to severe obesity. Obesity is a global health concern linked to chronic diseases ‚Äî this project leverages machine learning to identify obesity risk levels early, enabling proactive health interventions based on easily measurable lifestyle factors.

Dataset & Methodology

UCI Machine Learning Repository 2,111 Records ‚Ä¢ 17 Features

Features include: Age, Gender, Height, Weight, eating habits (vegetable consumption, high-calorie food intake), physical activity frequency, water intake, alcohol consumption, smoking status, transportation mode, and family history of overweight.

Target Variable (7 Classes)

Insufficient Weight | Normal Weight | Overweight Level I & II | Obesity Type I, II & III

Methodology

  • Data Preprocessing ‚Äî Verified data quality (no missing values), encoded categorical target variable using Label Encoding, applied StandardScaler for feature normalization
  • Exploratory Data Analysis ‚Äî Generated correlation heatmaps, identified key predictive features, created visualizations comparing obesity levels against weight, height, physical activity, and vegetable intake
  • Feature Selection ‚Äî Selected 4 high-impact features: Height, Weight, Physical Activity Frequency (FAF), and Vegetable Consumption (FCVC)
  • Model Development ‚Äî Trained and evaluated 4 classification algorithms with 80/20 train-test split

Model Performance Results

Model Accuracy F1-Score
Support Vector Machine (Linear) 95.03% 94.99%
Decision Tree 94.56% 94.54%
Logistic Regression 88.89% 88.71%
K-Nearest Neighbors 86.05% 86.05%

Key Findings

  • Best Performer ‚Äî SVM with linear kernel outperformed all other models
  • Feature Importance ‚Äî Weight and Height showed the strongest correlation with obesity classification
  • Activity Correlation ‚Äî Physical activity frequency inversely correlates with obesity levels
  • Interpretability ‚Äî Decision Tree visualization revealed interpretable decision boundaries for clinical use
