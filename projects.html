About the App

Stya is an AI-powered iOS wardrobe management and styling app that helps users digitize their closet, receive personalized outfit recommendations, and visualize complete looks on their own body. Users can photograph clothing items which are automatically analyzed and categorized using computer vision and GPT-4. The app generates occasion-based outfit suggestions using the user's actual wardrobe, creates photorealistic outfit visualizations using generative AI, and provides weather-aware styling recommendations. It features a token-based monetization system, real-time push notifications when outfit visualizations are ready, and supports multiple authentication methods including Apple Sign-In and Google Sign-In.

About the Project

Rocket is a full-stack web application that enables users to create, manage, and optimize professional resumes tailored to specific job descriptions. The platform leverages OpenAI's GPT-4 to automatically generate ATS-optimized resumes, provide personalized career advice, and create cover letters. Users can upload existing resumes for parsing, store their career history, and generate polished PDFs using multiple LaTeX templates (Classic, Modern, Minimal). The application features secure authentication, real-time resume generation with progress tracking, and export capabilities in PDF and DOCX formats.
